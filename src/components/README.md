# Components

Komponen reusable lintas fitur.

- `ui`: komponen dasar seperti button, input, badge, table.
- `layout`: shell halaman seperti sidebar, topbar, dan container.

Utamakan nama yang jelas, susunan JSX yang mudah dipindai, dan komentar untuk
aturan bisnis yang tidak terlihat dari kode. File boleh melebihi 120 baris,
tetapi pemeriksaan proyek membatasi 200 baris agar tanggung jawabnya tetap fokus.
